# Usage

1. git clone

``` js
  git clone https://github.com/hanghae-3/crypto_chart.git
```

2. crypto_chart 폴더를 인터프린터나 컴파일러로 열기
3. 필요한 라이브러리 설치

```
npm install
```

4. 실행

```
npm run start
```
